Reflective Journal - What I Learned in PPW

Week 1: Introduction to Web Development

This week I learned the fundamentals of HTML and CSS. I was surprised by how powerful CSS can be in creating visually appealing layouts. The box model concept was initially challenging but practicing with different elements helped me understand it better.

Week 2: Responsive Design

Media queries were a game-changer for me. Learning how to make websites adapt to different screen sizes opened my eyes to the importance of mobile-first design. I created my first responsive layout and tested it on multiple devices.

Week 3: JavaScript Basics

JavaScript was more complex than I expected. The concept of variables, functions, and DOM manipulation took time to grasp. However, after building a simple interactive form, I started seeing how JavaScript brings websites to life.

Week 4: Project Work

Working on my first complete project helped me understand how all the pieces fit together. Debugging was frustrating at times, but solving each issue gave me a sense of accomplishment and deeper understanding.
